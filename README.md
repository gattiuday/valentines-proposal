A quick and beautiful interactive Valentine's Day proposal website.

## Features
- **Interactive Envelope**: Click to open.
- **Typewriter Effect**: Personal message types out.
- **Date Picker**: Choose a date activity.
- **"No" Button Dodge**: A playful button that runs away.
- **Celebration**: Confetti and success screen.

## Technologies
- HTML5, CSS3, JavaScript
- Tailwind CSS (via CDN)
- Google Fonts

## Usage
Just open `index.html` in your browser!
