A quick and beautiful interactive Valentine's Day proposal website.

## Features
- **Interactive Envelope**: Click to open.
- **Typewriter Effect**: Personal message types out.
- **Date Picker**: Choose a date activity.
- **"No" Button Dodge**: A playful button that runs away.
- **Celebration**: Confetti and success screen.

## Technologies
- HTML5, CSS3, JavaScript
- Tailwind CSS (via CDN)
- Google Fonts

## Live Demo
[Check it out here!](https://gattiuday.github.io/valentines-proposal/)

## Deployment
This project is automatically deployed to GitHub Pages using GitHub Actions. Any push to the `master` branch triggers a new deployment.

## Usage
### Local Development
1. Clone the repository.
2. Open `index.html` in your browser.
